Welcome to Glue Stick Bot

Glue Stick is a powerful, advanced Discord bot designed to help you keep important messages glued in your channels. Whether it's an announcement, a rule, or any message you want to keep visible at the bottom, Glue Stick ensures that it always stays the last message in the channel.

  • Gluey Messages: Keep any message pinned to the bottom of your channel without manually doing it. Also the bot will delete its previous message to keep the channel clean.
  • Slash Commands: Glue Stick features user-friendly slash commands, simplifying management without the need to memorize complex command prefixes. It also enables server admins to control access to commands without granting additional management permissions to members.
  • Multi-line Glue: Use the /gluepara command to glue multi-line messages to a channel.
  • Embed Messages: Create glued embed messages with the /glueembed command, allowing for rich content display.
