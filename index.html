Welcome to Glue Stick Bot

Glue Stick is a powerful, advanced Discord bot designed to help you keep important messages glued in your channels. Whether it's an announcement, a rule, or any message you want to keep visible at the bottom, Glue Stick ensures that it always stays the last message in the channel.

  1. Invite Glue Stick: Click here to invite Glue Stick to your server!
  2. Set Up Your Gluey Message: Use the /glue command to set your first Gluey message in any channel.
  3. Manage Your Gluey Messages:
    • /glue [message]: Stick a new message in the channel.
    • /unglue: Remove the current sticky message in the channel.
    • /clearallglues: Remove all sticky messages from all channels.

ðŸ’¡ Pro Tips

  • One Gluey Per Channel: Glue Stick allows only one gluey message per channel. If you try to glue another, you'll replace the current glued message with the new one. No need to unglue the previous message first!
  • Auto-Refresh: Glue Stick will automatically delete and re-send the glued message every time new messages are posted, ensuring it stays at the bottom.
