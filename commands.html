Glue Stick Bot Commands

  • /glue - Glue a message to a channel
  • /glueembed - Glue an embed message to a channel
  • /unglue - Remove a glued message from a channel
  • /listglues - List all glued messages in the server
  • /clearallglues - Remove all glued messages in the server
  • /invite - Get an invite link for Glue Stick
  • /vote - Get a link to vote for Glue Stick on top.gg
  • /support - Get an invite to the Glue Stick support server

For more detailed information on each command, please refer to our documentation in the support server.
